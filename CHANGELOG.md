# Changelog

Все значимые изменения проекта Project CORE.

## [1.3.0] — 2026-06-18 — «Сбой сигнала»

Углубление эффекта разрыва реальности: звук, мозаичный распад, оживление узлов.

### Добавлено / Изменено
- **Глич — кинематографичный распад в пиксель-мозаику** (`GlitchFX`): фон деформируется и рассыпается мозаикой со смещением пластов-данных (data-slabs) и жёсткой RGB-каймой; приглушённая палитра вместо «мультяшного» зелёного. Звёзды просто исчезают (без разлёта).
- **Звук сбоя** (`lib/sfx.js`, Web Audio): процедурные радиопомехи + цифровой скрежет искажённой квадратной волны + падающие «эррор-бипы» на каждый разрыв. Включается тумблером звука.
- **«Другой мир»** во время кода — зелёные монохромные образы планеты и орбитальной системы за падающим кодом (~1.5с).
- **Голос** — хриплый компьютерный «грандж»: ниже pitch, медленнее, с эфирной подложкой радиопомех и ring-mod тоном вокруг фраз.
- **Узлы** — подписи стянуты к центру экрана (в зону видимости), указательные линии сохранены.
- **Превью в квадратах узлов** (`NodePreview`) — реалистичные «приборные» анимации по теме каждого узла (граф знаний, аналитика, трейдинг-свечи, IDE, 3D-каркас, осциллограф/спектр, таймлайн монтажа, изо-миникарта).

## [1.2.0] — 2026-06-18 — «Разрыв реальности»

Итерация кинематографичной полировки первого экрана. Доработки ведутся на ветке `v1-cinematic`.

### Добавлено / Изменено
- **Глич = «разрыв реальности»** — звёздный фон физически рассыпается (3D-разлёт точек через вершинный шейдер `uShatter` + `uFade`) и на ~1с заменяется падающим кодом, обратно тем же эффектом. Единый таймлайн в `app/page.js` синхронно дирижирует сценой и кодом; `MatrixRain` стал управляемым (props `active`/`glitch`).
- **Вспышка-ударная волна** (`.warp`) в момент разрыва — энергоимпульс вокруг планеты, маскирован, планету не задевает.
- **Скин планеты** во время глича плавно меняется со схематичного голубого на реалистичную текстуру Земли и возвращается (`Planet` prop `realistic`).
- **Код** ярче, читабельнее (JetBrains Mono 700, видимая «голова» капли, глубина по колонкам).
- **Звёзды** — гуще (~28 тыс.), мельче и реалистичнее; ~2/3 активно мерцают и переливаются (динамика яркости + лёгкий цветовой сдвиг).
- **Камера** — постоянная непрерывная скорость без остановок (убран sway-синус).
- **Ореол планеты** — мягче по цвету и прозрачнее.
- **Узлы тем** теперь сидят на реальных вершинах сетки-икосаэдра (farthest-point выборка), не выбиваются; русские подписи — «компьютерный» моноширинный JetBrains Mono.
- **Меню/заголовок** — «Веб дизайн» перенесён в правое меню; 4-словный список ровно под названием, стрелки в один вертикальный ряд слева.

## [1.1.0] — 2026-06-18 — «Киношная доработка v1»

Кинематографичная переработка первого экрана по референсам 001 (HUD-меню) и 002 (планета/звёзды/сетка). Ветка `v1-cinematic` от `v1.0.0`.

### Добавлено / Изменено
- **Планета** — реалистичная «Земля из точек» по 002: сэмплинг текстуры `earth-blue-marble.jpg` в точки по материкам, тёмное ядро для перекрытия задних точек, fresnel-атмосфера, освещение слева.
- **Звёзды** — реалистичное звёздное небо в реальном времени: 3 слоя глубины, натуральное распределение яркости, цвет по звёздной температуре (бело/голубо/тёпло-оранжевые), мягкие круглые точки без крестовых бликов, независимое мерцание каждой звезды.
- **Сетка вокруг планеты** — каркас-икосаэдр + плотная внутренняя решётка + светящиеся узлы-вершины + затухание по глубине (fog) + живой пульс яркости.
- **HUD-меню** по 001 — угловые неоновые плиты (циан + красный) сверху/снизу, инфографические полосы, золотой вертикальный список ENCOUNTER/RESOURCES/ORGANISATION/COMMUNITY с золотыми шевронами, заголовок PROJECT C.O.R.E.
- **Камера** — скользит по горизонту справа налево (без облёта на 360°), у линии горизонта.
- **Кинематографика** — лен-флер слева, ACES filmic tone mapping, плёночное зерно + виньетка.

### Технические заметки
- Текстуры (`public/textures/`) восстановлены из ветки `v2` (в `v1.0.0` отсутствовали).
- Стек: Next.js 14 (App Router) + React Three Fiber + three.js 0.160 + drei + Framer Motion.

# Test Landing Vercel — Project C.O.R.E.

Анимированный лендинг «Project C.O.R.E.» — космос-сцена с интерактивной сеткой узлов
(8 направлений), киберпанк-меню, матрица-эффект, процедурный dark-ambient звук и
голосовое сопровождение.

## Стек
- **Next.js 14** (App Router)
- **React Three Fiber + three.js + drei** — 3D-сцена первого экрана
- **Framer Motion** — анимации меню и блоков
- **Web Audio API** — процедурный dark-ambient (без аудиофайла)
- **Web Speech API** — женский голос (фразы Project CORE)

## Первый экран
Глубокий космос, мерцающие звёзды, вращающаяся планета, окружённая орбитальной
сеткой из нитей и узлов. 8 узлов интерактивны (Мышление, Маркетинг, Заработок,
Веб-дизайн, 3D, Аудио, Видео, Game) — при наведении подсвечиваются, показывают
описание и анимированное превью. Камера медленно облетает планету. Раз в 2–3 мин
фон ненадолго сменяется «матрицей» с глитч-переходом.

## Направления
Мышление · Маркетинг · Заработок · Веб-дизайн · 3D · Аудио · Видео · Game

## Запуск
```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # прод-сборка
```
